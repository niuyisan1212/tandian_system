## 📖 功能使用指南

### 一、店铺管理

#### 创建店铺
1. 点击右上角"新增店铺"按钮
2. 填写店铺名称
3. **在地图上选择位置**：
   - 方式一：搜索框输入地址搜索
   - 方式二：直接在地图上点击
   - 方式三：拖拽标记微调
4. 系统自动填充地址和经纬度
5. 填写其他信息（电话、营业时间等）
6. 点击"确定"保存

#### 编辑店铺
- 点击店铺卡片进入编辑模式
- 可修改所有信息包括位置

#### 删除店铺
- 点击店铺卡片下方的"删除"按钮
- 确认后永久删除

### 二、清单管理

#### 创建清单
1. 点击右上角"创建清单"按钮
2. 填写清单名称
3. 选择任务日期
4. **在地图上选择起点位置**：
   - 搜索地址
   - 地图点击选择
   - 拖拽微调
5. 从列表中选择要探店的店铺（可多选）
6. 点击"确定"保存
7. 系统自动规划最优路线

#### 查看清单详情
- 点击清单卡片查看详情
- 查看路线方案（时间优先/成本优先/均衡）
- 查看路线指引和交通方式

#### 完成清单
- 探店完成后点击"标记完成"
- 系统自动更新所有店铺状态为"已探店"

### 三、地图功能

#### 地图首页
- 查看所有待探店店铺位置
- 选择清单查看特定店铺
- 统计数据展示

#### 地图交互
- 滚轮缩放地图
- 拖拽平移地图
- 点击标记查看信息
- 自动适应视野
