## 三、功能使用说明

### 1. 地理编码
后端自动调用地理编码API，无需手动操作。

**触发场景**：
- 添加/编辑店铺时输入地址
- 系统自动解析为经纬度坐标

**降级策略**：
- API调用失败时使用模拟坐标（上海人民广场）

### 2. 路线规划
创建清单时自动生成3种路线方案：

**时间优先方案**：
- 优先使用地铁、快速公交
- 总耗时最短

**成本优先方案**：
- 优先使用普通公交
- 总花费最低

**均衡方案**：
- 综合考虑时间和成本
- 推荐步行（探店场景）

**距离策略**：
- ≤ 800米：步行
- 800-3000米：步行（非成本优先）
- > 3000米：公交/地铁

### 3. 地图展示

**地图首页**：
- 显示所有有效待探店店铺
- 点击标记查看店铺信息
- 选择清单后高亮显示相关店铺

**清单详情页**：
- 显示清单中的店铺位置
- 绘制规划路线
- 不同交通方式用不同颜色标识

```java
private RoutePlan createPlan(

// 规划两点之间的路线

```

### 前端核心组件

#### AMapComponent.vue
```vue
<AMapComponent
  container-id="mapId"
  :center-lng="121.4737"
  :center-lat="31.2304"
  :zoom="13"
  :markers="markerList"
  :route-path="routeList"
  @marker-click="handleMarkerClick"
  @map-ready="handleMapReady"
/>
```

**Props**：
- `containerId`: 地图容器ID
- `centerLng/centerLat`: 中心点坐标
- `zoom`: 缩放级别
- `markers`: 标记点数组
- `routePath`: 路线数组

**Events**：
- `map-click`: 地图点击
- `marker-click`: 标记点击
- `map-ready`: 地图准备完成

## 八、故障排查

### 问题1：地图不显示
**原因**：高德地图SDK未加载  
**解决**：检查网络连接，确认API Key正确

### 问题2：路线规划失败
**原因**：API调用失败或超出配额  
**解决**：检查控制台日志，系统会自动降级为模拟数据

### 问题3：坐标偏移
**原因**：坐标系不一致  
**解决**：确保使用GCJ-02坐标，其他坐标系需转换

### 问题4：标记点不显示
**原因**：经纬度数据格式错误  
**解决**：检查数据类型，确保为数字类型
