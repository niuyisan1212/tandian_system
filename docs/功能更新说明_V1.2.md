## 🚀 使用指南

### 1. 创建店铺

**步骤**：
1. 点击"新增店铺"按钮
2. 填写店铺名称
3. 在地图上搜索或点击选择位置
4. 系统自动填充地址和经纬度
5. 填写其他信息（电话、营业时间等）
6. 点击确定保存

**地图选择方式**：
- 方式一：在搜索框输入地址搜索
- 方式二：直接在地图上点击选择
- 方式三：拖拽标记微调位置

### 2. 创建清单

**步骤**：
1. 点击"创建清单"按钮
2. 填写清单名称
3. 选择任务日期
4. 在地图上选择起点位置
5. 从列表中选择要探店的店铺
6. 点击确定保存
7. 系统自动规划路线

## 📊 技术实现

### 前端技术栈
- Vue 3.4 Composition API
- 高德地图 JS API 2.0
- Ant Design Vue 4.0
- Day.js日期处理

### 核心组件

#### LocationPicker
**功能**：地图位置选择器

**Props**：
- `initLng`: 初始经度
- `initLat`: 初始纬度
- `initAddress`: 初始地址

**Events**：
- `select`: 选择位置时触发

**Methods**：
- `setLocation(lng, lat, address)`: 设置位置

### 地图功能

#### 搜索功能
```javascript
AMap.plugin('AMap.PlaceSearch', () => {
  const placeSearch = new AMap.PlaceSearch({
    pageSize: 10,
    pageIndex: 1,
    city: '上海'
  })
  placeSearch.search(keyword, callback)
})
```

#### 逆地理编码
```javascript
AMap.plugin('AMap.Geocoder', () => {
  const geocoder = new AMap.Geocoder({
    radius: 1000,
    extensions: 'all'
  })
  geocoder.getAddress([lng, lat], callback)
})
```

#### 地图点击
```javascript
map.on('click', (e) => {
  const lng = e.lnglat.getLng()
  const lat = e.lnglat.getLat()
  // 处理点击事件
})
```
