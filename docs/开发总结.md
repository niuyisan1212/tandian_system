## 二、技术实现说明

### 环境适配
- **原计划**: Spring Boot 3.x + Java 17
- **实际实现**: Spring Boot 2.7.18 + Java 1.8
- **原因**: 当前系统Java版本为1.8

### 数据库设计
- 数据库：tandian_system
- 表数量：4个（shop, shopping_list, list_shop, route_plan）
- 符合第三范式
- 包含完整的外键约束和索引

### 路线规划算法
- 使用最近邻算法解决TSP问题
- 根据距离选择交通方式（步行/公交/地铁）
- 支持3种策略方案生成

### 地图集成
- **当前**: 使用模拟数据展示
- **预留**: 高德地图API接口
- **配置**: application.yml中配置API Key

## 四、待确认事项

### 1. 高德地图API（需申请）
**说明**: 首版使用模拟数据，实际部署需申请高德地图API Key  
**解决方案**:
- 方案一：申请个人开发者Key（免费配额足够）
- 方案二：申请企业开发者Key（更高配额）

**配置方式**:
```yaml
# backend/src/main/resources/application.yml
amap:
  api-key: YOUR_AMAP_KEY
  enabled: true
```

### 2. 数据库密码安全（建议优化）
**说明**: 配置文件中密码明文存储  
**解决方案**:
- 方案一：使用环境变量
- 方案二：使用Jasypt加密配置

**推荐**: 使用环境变量
```bash
export DB_PASSWORD=niuhaoqing1212
java -jar system-1.0.0.jar
```

### 3. 生产环境部署（需规划）
